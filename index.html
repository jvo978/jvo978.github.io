Auto Repair Tracker

A car repair tracking application that displays the cars that need repair along with the car information. Supports user authentication and all CRUD actions (create, read, update, and delete cars). Glumon

A glucose monitoring application that tracks patient glucose levels under specific doctors. Front End was built with HTML, CSS, JavaScript, Ember.js, Handlebar.js, AJAX, and Sass. Back End was built with Ruby on Rails and PostgreSQL. Supports user authentication and all CRUD actions (create, read, update, and delete doctors). I chose this topic for the application because my mom is a diabetic, and it is very important to monitor your glucose level when you have this health condition. Status: In the process of adding patients to application and all CRUD actions (create, read, update, and delete patients).
